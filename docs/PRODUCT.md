# Product Definition

## Primary Workflow

Story Room
→ Contributions
→ Staff Review
→ Memory Cards
→ Story Map
→ Guided Session
→ Story Capsule
→ Delivery

## User Roles

### Owner

Starts the Story Room and drives the family project.

### Contributor

Adds memories, photos, voice notes, questions, recipes, documents, captions, stories, and artifacts.

### Staff

Turns raw material into structured production assets.

## Main Production Objects

### Story Room

Operational center for a single family story project.

### Contribution

Raw memory material submitted by family or contributors.

### Memory Card

Structured story block extracted from a contribution.

### Story Map

Interview and production blueprint.

### Story Capsule

Final deliverable.

## Product Rule

Every screen must help move the Capsule toward completion.

## Product Philosophy

The product is not the software.

The product is the transformation:

Scattered memory fragments
→
Finished intergenerational artifact.
