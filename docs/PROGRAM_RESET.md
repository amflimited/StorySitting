# Program Reset

The prior work kept improving the marketing site, but the actual product system was not moving fast enough.

This package is the correction.

The business does not advance because the homepage is prettier. The business advances when:

- a family can create a private Story Room
- relatives can submit material
- all inputs normalize into Contributions
- staff can review and organize them
- a Story Map can be produced
- a Story Capsule can be delivered

The next launch gate is not another design package.

The next launch gate is:

Owner → Story Room → Invite → Contribution → Staff Review → Memory Card → Story Map
